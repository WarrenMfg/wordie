# wordie

CLI dictionary and thesaurus using [Datamuse](https://www.datamuse.com/).

## Usage with `npm`

**Installation**

`npm install -g wordie`

**Example**

`wordie coffee`

## Usage with `npx`

**Installation**

No installation.

**Example**

`npx wordie coffee`
